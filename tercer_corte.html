PROGRAMACION CONCURRENTE

La programación concurrente permite desarrollar software que ejecuta eventos o circunstancias que están sucediendo o existen al mismo tiempo

Hilo

Son unidades de ejecución más pequeñas dentro de un proceso que pueden ejecutarse de forma independiente entre sí. Permiten que un programa realice múltiples tareas simultáneamente, dividiendo su ejecución en partes más pequeñas que pueden ejecutarse en paralelo.

        
          public class EjemploHilo implements Runnable {
    
            public void run() {
                // El código que se ejecutará en el hilo
                for (int i = 0; i < 5; i++) {
                    System.out.println("Hilo en ejecución: " + i);
                    try {
                        Thread.sleep(1000); // Espera de 1 segundo
                    } catch (InterruptedException e) {
                        System.out.println("El hilo fue interrumpido.");
                    }
                }
            }
            
            public static void main(String[] args) {
                // Crear una instancia del objeto que implementa Runnable
                EjemploHilo ejemploHilo = new EjemploHilo();
                
                // Crear un objeto Thread y pasarle la instancia Runnable
                Thread hilo = new Thread(ejemploHilo);
                
                // Iniciar la ejecución del hilo
                hilo.start();
                
                // Puedes continuar ejecutando más código aquí mientras el 
                hilo se está ejecutando en segundo plano
                
                try {
                    // Esperar a que el hilo termine antes de que el 
                    programa principal termine
                    hilo.join();
                } catch (InterruptedException e) {
                    System.out.println("El hilo principal fue interrumpido.");
                }
                
                System.out.println("El programa ha finalizado.");
            }
        }

En este ejemplo, creamos una clase EjemploHilo que implementa la interfaz Runnable. El método run() contiene el código que será ejecutado por el hilo. En el método main, creamos un nuevo hilo (Thread) pasándole una instancia de EjemploHilo. Luego, iniciamos el hilo con hilo.start() y esperamos a que el hilo termine su ejecución con hilo.join() antes de que el programa principal finalice.
